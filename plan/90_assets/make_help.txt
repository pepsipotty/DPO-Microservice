DPO Microservice Development Commands
=====================================

Setup and Installation:
  setup          - Complete development environment setup
  install        - Install Python dependencies

Development:
  test           - Run all tests
  test-api       - Run API integration tests
  lint           - Run code linting (if tools available)
  validate-structure - Validate package imports and structure

Training Pipeline:
  toy-data       - Generate toy dataset for testing
  toy-train      - Run training with toy data
  toy-trigger    - Test full API â†’ training pipeline

API Server:
  api            - Start the FastAPI webhook server

Docker:
  docker-build   - Build Docker image
  docker-run     - Run Docker container
  docker-clean   - Clean Docker artifacts

Cleanup:
  clean          - Clean temporary files and caches
